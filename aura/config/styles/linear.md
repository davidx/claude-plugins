# Linear Style

Ultra-clean, productivity-focused aesthetic inspired by Linear.app

## Characteristics
- Monochromatic with purple/violet accents
- Dark mode friendly (provide both)
- Keyboard-first feel
- Dense but not cramped information display
- Status indicators with color coding
- Command palette / search prominent

## Colors (Dark)
- Background: #0D0D0D, #141414
- Surface: #1C1C1C, #232323
- Border: #2E2E2E
- Text: #FFFFFF, #A3A3A3
- Primary: #5E5ADB (violet)
- Success: #22C55E
- Warning: #EAB308
- Error: #EF4444

## Colors (Light)
- Background: #FFFFFF
- Surface: #F5F5F5
- Border: #E5E5E5
- Text: #171717, #525252
- Primary: #5E5ADB

## Typography
- Font: Inter
- Weights: 400, 500 only
- Sizes: 13px base, tight scale

## Signature Elements
- Rounded-md corners (6px)
- Subtle shadows in light mode
- Keyboard shortcuts displayed inline
- Compact tables/lists
- Minimal iconography
