# Minimal Style

Extreme simplicity, content-first design

## Characteristics
- Maximum whitespace
- Black and white with one accent
- Typography-driven hierarchy
- No shadows, minimal borders
- Large touch targets
- Single column layouts preferred

## Colors
- Background: #FFFFFF
- Text: #000000, #666666
- Accent: User choice or #000000
- Borders: #E0E0E0 (sparingly)

## Typography
- Font: System stack or Inter
- Large base size (16-18px)
- High contrast weights (400 vs 700)
- Generous line-height (1.7+)

## Signature Elements
- Underline links, not color
- Outlined buttons only
- No cards, use spacing
- Horizontal rules for separation
